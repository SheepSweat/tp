# ModTrack

Ever wanted to know what modules you have taken so far in your computer engineering course? Well now you can keep track of it easily using ModTrack!
ModTrack is a CLI-based NUS modules tracking system that allows NUS students to keep track of their completed modules and compare it with their graduation requirements.
